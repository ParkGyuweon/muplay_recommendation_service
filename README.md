### MuPlay : 사용자 맞춤 뮤지컬, 연극 추천 서비스

### 프로젝트 소개
- MuPlay는 사용자가 회원가입 시 설정한 취향 뮤지컬, 연극과 가장 유사성이 높은 공연을 추천해주는 서비스입니다. 
- 현재 진행 중인 뮤지컬과 연극 안에서 추천이 이루어지도록 NOL, 티켓링크, 예스24, 네이버 예매의 데이터를 매일 받아옵니다.
- 예매 사이트에 등록된 시놉시스 뿐만 아니라 관람객의 진솔한 후기를 기반으로 추천이 이루어질 수 있도록 네이버 블로그, X 등의 데이터를 사용합니다.
- 해당 서비스는 공연 자체를 좋아하는 연뮤덕을 위한 서비스로, 공연은 보고 싶지만 볼 공연이 마땅치 않은 분들이 대학로 길거리에서도 바로 쓸 수 있도록 만드는 것이 목적입니다.

### 기술 스택
JDK 17, spring boot, react, postgreSQL, GIT
